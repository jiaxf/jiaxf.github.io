Mac下零配置server

PowPow is a zero-config Rack Server for Mac。

Install

curl get.pow.cx | sh

Setup

Symlink the folder into ~/.pow

$ cd ~/.pow $ ln -s /path/to/myapp

Your website will be up and running at http://myapp.dev. The URL is based on the name of the symlink.
